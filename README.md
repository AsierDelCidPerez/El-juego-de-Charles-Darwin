# El-juego-de-Charles-Darwin


El juego de Charles Darwin ha sido creado por Asier Del Cid Pérez.
Es recomendable hacer lectura del siguiente documento para una correcta instalación del juego y por disfrutarlo al máximo
El juego de Charles Darwin ha sido creado en el lenguaje de programación Java y compilado en la versión Java SE 9

Requisitos:
	- Java SE 9 o posterior (Si no lo tienes instalado: https://www.oracle.com/es/java/technologies/javase/javase9-archive-downloads.html)

Instalación:
	- Instala el archivo "Juego Darwin.jar" adjunto
	- Ábrelo
	- Disfruta del juego

El juego se reserva la licencia estándar Apache 2.0 de programación estándar.

Para saber más información sobre el proyecto o de la licencia le invito a que visite mi github:
https://github.com/AsierDelCidPerez/El-juego-de-Charles-Darwin
